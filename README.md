# Практична робота 10.2

## Тема: Async/Await та робота з API

### Варіант 7: Country Explorer

Реалізовано:

- завантаження списку країн через REST Countries API
- пошук країни за назвою
- відображення прапора, столиці, регіону та населення
- async/await
- fetch API
- try/catch
- loading state
- error handling

### API

REST Countries API

### Технології

- JavaScript
- HTML
- CSS

### Запуск

1. Відкрити проєкт у VS Code
2. Запустити через Live Server
3. Використовувати пошук або кнопку "Показати всі країни"
